## About Me

I'm a Data Annotation Specialist with 2.5 years at Eligarf Technologies, Pune, working across data annotation, quality assurance, and data analysis.

I write SQL for exploratory analysis and reporting, use Python (Pandas, NumPy, Matplotlib) for data cleaning and visualization, and build dashboards in Power BI
and Excel. I've worked with annotation tools including Superannotate, CVAT, Labelbox, and Labelme across image, video, and text datasets.

Notable work:

- Cut defect counts by 25% via root cause analysis
- Built a Customer Behavior Analytics Dashboard using SQL and Python
- Published research paper on JARVIS — a bilingual Python voice assistant

B.E. in Information Technology | Savitribai Phule Pune University Certified in Data Analytics with Python

Currently open to Data Analyst, Business Analyst, and AI/ML Data roles in Pune, Mumbai, Kochi, or remote.
